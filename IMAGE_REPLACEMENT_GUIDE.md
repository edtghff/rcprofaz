# Руководство по замене изображений

## Структура изображений на сайте

Все изображения находятся в папке `public/images/`:

```
public/images/
├── logo.png                    # Логотип компании (используется в Header и Footer)
├── logo.jpeg                   # Альтернативный логотип
├── hero-banner.jpg             # Главный баннер на главной странице
├── about-preview.jpg           # Изображение в секции "О нас" на главной странице
├── about-main.jpg              # Главное изображение на странице "О нас"
│
├── services/                   # Изображения для услуг
│   ├── tikinti-temir-isleri.jpg
│   ├── dizayn-layihelendirme.jpg
│   ├── lift-satisi-servisi.jpg
│   ├── qapi-sistemleri.jpg
│   ├── suse-cam-balkon-sistemleri.jpg
│   └── suse-satisi.jpg
│
├── products/                   # Изображения для продуктов
│   ├── liftler.jpg
│   ├── eskalatorlar.jpg
│   ├── insaat-avadanliqlari.jpg
│   └── avtomatik-giris.jpg
│
├── projects/                   # Изображения для проектов
│   ├── project-1.jpg
│   ├── project-2.jpg
│   ├── project-3.jpg
│   └── project-4.jpg
│
└── news/                       # Изображения для новостей
    ├── news-1.jpg
    └── news-2.jpg
```

## Как заменить изображения

### Вариант 1: Простая замена файлов

1. **Подготовьте новые изображения** с теми же именами файлов
2. **Замените файлы** в соответствующих папках:
   - Просто скопируйте новые изображения в `public/images/` (или подпапки)
   - Убедитесь, что имена файлов точно совпадают

### Вариант 2: Использование скрипта

Запустите скрипт `replace-images.sh` (см. ниже)

### Вариант 3: Ручная замена через Finder/Explorer

1. Откройте папку `public/images/` в Finder (Mac) или Explorer (Windows)
2. Найдите нужное изображение
3. Замените его новым файлом с тем же именем

## Требования к изображениям

- **Формат**: JPG, JPEG или PNG
- **Рекомендуемые размеры**:
  - Логотип: 200-400px ширина, прозрачный фон (PNG)
  - Баннеры: 1920x1080px или больше
  - Изображения услуг/продуктов: 800x600px или больше
  - Изображения проектов: 1200x800px или больше
- **Оптимизация**: Рекомендуется оптимизировать изображения перед загрузкой (можно использовать онлайн-инструменты типа TinyPNG)

## Где используются изображения

### Логотип
- `components/Header.tsx` - шапка сайта
- `components/Footer.tsx` - подвал сайта

### Главная страница
- `app/page.tsx`:
  - `hero-banner.jpg` - главный баннер
  - `about-preview.jpg` - секция "О нас"

### Страница "О нас"
- `app/haqqimizda/page.tsx`:
  - `about-main.jpg` - главное изображение

### Услуги
- `app/xidmetler/[slug]/page.tsx`:
  - Автоматически использует `/images/services/{slug}.jpg`
  - Например: `tikinti-temir-isleri.jpg` для услуги с slug `tikinti-temir-isleri`

### Продукты
- `data/productsData.ts` - определяет пути к изображениям
- `app/mehsullarimiz/[slug]/page.tsx` - отображает изображения

### Проекты
- `data/projectsData.ts` - определяет пути к изображениям
- `app/layiheler/page.tsx` - список проектов
- `app/layiheler/[slug]/page.tsx` - детали проекта

### Новости
- `app/xeberler/page.tsx` - список новостей
- `app/xeberler/[slug]/page.tsx` - детали новости

## После замены изображений

1. **Перезапустите dev-сервер** (если он запущен):
   ```bash
   npm run dev
   ```

2. **Очистите кеш браузера** (Ctrl+Shift+R или Cmd+Shift+R)

3. **Проверьте изображения** на сайте

## Примечания

- Изображения автоматически оптимизируются Next.js через компонент `next/image`
- Если изображение не найдено, Next.js покажет ошибку в консоли браузера
- Убедитесь, что новые изображения имеют правильные расширения (.jpg, .jpeg, .png)
